## D — Что ещё сделать (приоритет)

### Блокируется бэкендом (сначала API в сервисах + Traefik)

| Приоритет | Задача |
|-----------|--------|
| P1 | **Склад:** GET (и при необходимости POST) в `warehouse` → `ApiPaths` + провайдер + таблица вместо заглушек |
| P1 | **Логистика:** список рейсов/маршрутов в `logistics` → то же |
| P2 | **Заказы:** POST создание заказа с UI «Новый заказ»; серверная пагинация / фильтр «за сегодня» для точного KPI |
| P2 | **Админ:** список пользователей — эндпоинт(ы) в `identity` (или отдельный BFF), затем таблица на `/admin` |

### Только фронт (можно параллельно)

| Приоритет | Задача |
|-----------|--------|
| P2 | **Каталог:** локальный поиск/фильтр по уже загруженному списку (опционально отдельный запрос, если появится в API) |
| P2 | **Заказы:** пагинация на UI при `skip/limit` на бэке |
| P3 | **Скелетоны** и единый вид пустых списков |
| P3 | **JWT:** перевести вызовы с `rawDio` на `dioProvider`, когда микросервисы начнут требовать Bearer |
| P4 | **Дашборд:** лента активности — нужен источник (очередь событий, REST агрегат или WebSocket) |
| P4 | **Persist:** `shared_preferences` для токенов + при старте `refreshProfile()` / редирект по `AUTH_ENABLED` |

## Флаги сборки

| Флаг | Назначение |
|------|------------|
| `API_BASE_URL` | База Traefik (на Web по умолчанию — `Uri.base.origin`) |
| `AUTH_ENABLED=true` | Обязательный вход, старт с `/login` |
