# Дорожная карта фронтенда GastroRoute

Актуальный чеклист: **`docs/PLAN_TASKS.md`**.  
Продуктовый UI (без dev-текстов, рабочие кнопки): **`docs/PLAN_UI_PRODUCT.md`**.

## Уже реализовано

- Каталог: список, карточка, глобальный поиск, **создание товара**.
- Заказы: список, детали, **создание заказа**.
- Финансы: баланс и транзакции; **счёт и оплата**; `client_id` синхронизируется с профилем после входа.
- Склад: остатки `GET`, **оприходование**.
- Логистика: список маршрутов, **создание рейса**; API поддерживает **фильтр `status`** для списка.
- Дашборд: KPI (каталог, заказы сегодня, выручка за день по транзакциям, **рейсы в пути по серверному `total`**); частичные ошибки; pull-to-refresh.
- Админ: список пользователей, **регистрация**.
- Ошибки сети: `dioErrorMapper`; интерфейс без упоминаний Traefik/dev-команд.
- Вход по `AUTH_ENABLED`, профиль, GoRouter.
- **Фаза 2:** persist access/refresh, восстановление сессии и `GET /users/me` после старта; каталог/заказы/склад/финансы/логистика через `dioProvider` (Bearer при наличии токена).
